# Yandex AI Studio Cookbook

Cookbook — это сборник практических рецептов и готовых примеров (end-to-end) для создания AI-приложений и агентов с помощью [Yandex AI Studio](https://aistudio.yandex.ru/).

## Создание агентов

В Yandex AI Studio вы можете создавать, тестировать и масштабировать собственных AI-агентов для решения бизнес-задач.

👉 **[Создать агента в Yandex AI Studio (Ателье агентов)](https://aistudio.yandex.ru/agent-atelier)**

В Ателье агентов доступны встроенные инструменты (веб-поиск, поиск по файлам, интерпретатор кода), интеграции через MCP-серверы и удобный интерфейс для прототипирования.

## Примеры в репозитории

В данном репозитории собраны следующие примеры:

1. **MosEx News Monitoring Agent (`stock_agent`)**  
   Агент для мониторинга новостей Московской биржи с использованием веб-поиска.

2. **Yandex Cloud Competitor Monitoring (`competitor_monitoring`)**  
   Система мониторинга конкурентов с применением поиска, интерпретатора кода (Code Interpreter) и облачных функций.

3. **Yandex Cloud Onboarding Agent (`onboarding_agent`)**  
   Агент для помощи в онбординге пользователей.

4. **Yandex Cloud Multi-Agent System (`multi_agent`)**  
   Многоагентная система на базе LangGraph, использующая Vector Store и поиск по файлам.
