# Рецепт: Модульные Системные Промпты

Вместо одного огромного `system_prompt`, который сложно поддерживать, мы используем **модульный подход**. Промпт собирается динамически из «кирпичиков» под конкретную бизнес-задачу (например, «строгий дайджест для CEO» или «творческий поиск трендов»).

### 1. Базовое ядро (Core)

Этот блок задает роль и базовые правила безопасности, которые активны всегда.

```python
# --- БАЗОВЫЙ ПРОМПТ (CORE) ---
base_system_prompt = """
РОЛЬ И ЦЕЛЬ:
Ты — аналитический агент мониторинга новостей и публичных данных.
Твоя задача — находить актуальные публикации через web_search, извлекать факты, 
очищать шум и формировать аналитический вывод для принятия решений.

БАЗОВЫЕ ПРИНЦИПЫ:
1. Фактичность: опирайся строго на источники. Не выдумывай данные.
2. Прозрачность: разделяй "факт" и "интерпретацию".
3. Проверяемость: всегда сохраняй ссылки на источники.
4. Нейтральность: используй деловой стиль без маркетинговых клише.

ИНСТРУКЦИЯ WEB_SEARCH:
- Используй поиск для проверки фактов и получения свежих данных.
- Поиск производится по ключевым словам (тикеры, события, регионы).
- Если данных недостаточно — сообщи об этом прямо.
"""
```

### 3. Конструктор промпта (Builder)

Функция, которая собирает финальный промпт под сценарий.

```python
def build_system_prompt(task_type="default"):
    """
    Собирает системный промпт из модулей под конкретную задачу
    """
    # 1. Ядро (всегда включено)
    blocks = [
        base_system_prompt,
        addon_mandatory_citations,  # Всегда требуем ссылки
        addon_compliance_no_pii,    # Всегда защищаем PII
        addon_no_assumptions        # Всегда запрещаем выдумки
    ]
    
    # 2. Сценарии
    if task_type == "digest_boss":
        # Строгий отчет для руководства
        blocks.extend([
            addon_recency_30d,
            addon_sources_official_only,
            addon_compliance_fin_legal,
            addon_postprocess_exec_digest,
            addon_output_markdown
        ])
        
    elif task_type == "market_research":
        # Глубокое исследование рынка
        blocks.extend([
            addon_external_analytics_trends,
            addon_external_analytics_market_aggregation,
            addon_output_markdown
        ])
        
    elif task_type == "compare_companies":
        # Сравнение конкурентов
        blocks.extend([
            addon_external_analytics_compare_companies,
            addon_output_markdown
        ])
        
    else:
        # Default: обычный поиск
        blocks.append(addon_output_markdown)

    return "\n\n".join(blocks)
```


### 4. Пример использования

```python
# Создаем промпт для генерации отчета руководителю
system_prompt = build_system_prompt("digest_boss")

# Инициализация агента

)
```
