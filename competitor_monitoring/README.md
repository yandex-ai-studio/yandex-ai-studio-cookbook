# Агент пакетного мониторинга конкурентов с оркестрацией
**Агент**, который обходит N конкурентов через *WebSearch Responses API*, собирает новости и изменения цен, генерирует сводный дайджест: обрабатывает конкурентов параллельно, логирует каждый шаг и визуализирует результаты через *Code Interpreter*. Весь пайплайн: параллельный сбор данных с retry, трейсинг, агрегация в дайджест.

## Best Practices

*   **Выбор модели**: Используйте `DeepSeek` для аналитических задач с фактами из WebSearch, `YandexGPT Pro` для генерации дайджестов.
*   **Промпт-инжиниринг**: Детально описывайте роль агента, формат ответа и ограничения в системном промпте. Используйте температуру 0.2–0.3.
*   **Параллелизм**: Используйте `asyncio.Semaphore` для контроля числа одновременных запросов к API.
*   **Устойчивость**: Добавляйте retry с exponential backoff только для retryable-ошибок (429, 500, 502, 503, 504).
*   **Структурированный вывод**: Применяйте Pydantic-модели для предсказуемого JSON от модели.
*   **Безопасность**: Храните API-ключи в `.env` файле, никогда не коммитьте их в репозиторий.
