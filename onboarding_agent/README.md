# Навигация: Онбординг-агент для освоения сервисов Yandex Cloud
**Агент‑наставник** с *WebSearch Tool*, который помогает новым сотрудникам и юзерам быстро осваивать документацию *Yandex Cloud* и сохранять ее в *Object Storage*, формируя собственную базу знаний. Весь пайплайн: находит актуальную документацию и примеры через *WebSearch Responses API*, сохраняет в *Object Storage* полезные материалы.

## Best Practices

1. **Промпт-инжиниринг**
   - Тестируйте разные варианты system prompt
   - Используйте температуру 0.2-0.3 для детерминированности
   - Явно указывайте требуемый формат ответа

2. **Обработка ошибок**
   - Ловите NetworkError при проблемах с API
   - Логируйте все ошибки для отладки

3. **Оптимизация производительности**
   - Используйте Lite модель для простых запросов
   - Ограничивайте max_output_tokens для быстроты
   - Кешируйте результаты частых запросов
   - Используйте `rate_limit` и `retry`

4. **Безопасность**
   - Никогда не коммитьте API ключи
   - Используйте .env файлы
   - Регулярно ротируйте ключи доступа

5. **Масштабирование**
   - Используйте асинхронные запросы для параллелизма
   - Добавьте rate limiting
   - Мониторьте использование токенов
